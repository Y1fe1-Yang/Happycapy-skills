# Image Enhancer

Improves the quality of images, especially screenshots, by enhancing resolution, sharpness, and clarity. Perfect for preparing images for presentations, documentation, or social media posts.

## Installation

```bash
mkdir -p ~/.claude/skills
cp -r image-enhancer ~/.claude/skills/
```

## Usage

Use this skill when you need to:
- Improve screenshot quality for blog posts or documentation
- Enhance images before sharing on social media
- Prepare images for presentations or reports
- Upscale low-resolution images
- Sharpen blurry photos
- Clean up compressed images
