# Mobile Design

Mobile-first design thinking and decision-making for iOS and Android apps. Touch interaction, performance patterns, platform conventions. Teaches principles, not fixed values.

## When to Use

Use when building React Native, Flutter, or native mobile apps. This skill provides comprehensive guidance on:

- Touch-first design principles
- Platform-specific conventions (iOS & Android)
- Mobile performance optimization
- Navigation patterns
- Mobile UX psychology
- Framework decision-making
- Security best practices

## Key Features

- **Anti-Memorization Approach**: Forces context-based thinking instead of AI defaults
- **Platform-Specific Guidance**: Separate iOS and Android conventions
- **Performance-Focused**: React Native and Flutter optimization patterns
- **Touch Psychology**: Fitts' Law, thumb zones, gesture design
- **Decision Trees**: Framework, state management, and storage selection
- **Comprehensive Reference Files**: 13 detailed guides covering all aspects of mobile development

## Reference Files

- `mobile-design-thinking.md` - Anti-memorization, forces thinking (READ FIRST!)
- `touch-psychology.md` - Fitts' Law, gestures, haptics, thumb zone
- `mobile-performance.md` - RN/Flutter performance, 60fps, memory
- `mobile-backend.md` - Push notifications, offline sync, mobile API
- `mobile-testing.md` - Testing pyramid, E2E, platform-specific
- `mobile-debugging.md` - Native vs JS debugging, Flipper, Logcat
- `mobile-navigation.md` - Tab/Stack/Drawer, deep linking
- `mobile-typography.md` - System fonts, Dynamic Type, accessibility
- `mobile-color-system.md` - OLED, dark mode, battery-aware
- `platform-ios.md` - Human Interface Guidelines, SF Pro, SwiftUI
- `platform-android.md` - Material Design 3, Roboto, Compose
- `decision-trees.md` - Framework/state/storage selection
- `touch-psychology.md` - Touch interaction principles

## Scripts

- `scripts/mobile_audit.py` - Mobile UX & Touch Audit

## Core Principles

- Touch-first design
- Battery-conscious development
- Platform-respectful implementation
- Offline-capable architecture
- Mobile is NOT a small desktop
