# Video Downloader

Downloads videos from YouTube and other platforms for offline viewing, editing, or archival. Handles various formats and quality options.

## Installation

```bash
mkdir -p ~/.claude/skills
cp -r video-downloader ~/.claude/skills/
```

## Usage

Use this skill when you need to:
- Download YouTube videos for offline viewing
- Save educational content for reference
- Archive important videos
- Get video files for editing or repurposing
- Download your own content from platforms
- Save conference talks or webinars
