# Data Storytelling Skill

Transform data into compelling narratives using visualization, context, and persuasive structure.

## Description

This skill helps you create data-driven stories that drive decisions and inspire action. It provides frameworks, templates, and best practices for presenting analytics to stakeholders, creating business reviews, and building executive presentations.

## When to Use

- Presenting analytics to executives
- Creating quarterly business reviews
- Building investor presentations
- Writing data-driven reports
- Communicating insights to non-technical audiences
- Making recommendations based on data

## Key Features

- **Story Frameworks**: Problem-Solution, Trend, and Comparison story structures
- **Visualization Techniques**: Progressive reveal, contrast and compare, annotation and highlighting
- **Presentation Templates**: Executive summary slides, data story flows, and one-page dashboards
- **Writing Techniques**: Effective headlines, transition phrases, and handling uncertainty

## Installation

```bash
mkdir -p ~/.claude/skills
cp -r data-storytelling ~/.claude/skills/
```

## Usage

Activate the skill when working on data presentations or analytics reports:

```
Use the data-storytelling skill to help me create a quarterly business review presentation.
```

## Examples

### Example 1: Customer Churn Analysis

Create a problem-solution story that identifies patterns in customer churn and provides actionable recommendations with ROI calculations.

### Example 2: Performance Trend

Build a trend story showing transformation over time with before/after comparisons and key insights about what drove the change.

### Example 3: Market Opportunity

Develop a comparison story to evaluate options using weighted scoring matrices and clear recommendations.
